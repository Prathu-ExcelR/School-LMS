# School-LMS

Learning Management System built with React, Node.js, Express, and MySQL.

## Features
- Student Dashboard
- Course Management
- Progress Analytics
- Quiz System
- Authentication

## Tech Stack
- Frontend: React + Vite + Tailwind CSS
- Backend: Node.js + Express
- Database: MySQL

## Setup

### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
npm run dev
```

Configure MySQL database using `server/schema.sql` and update `server/.env` with your credentials.
